About the Project

Bridging Theory &
Hands-On Robotics

An all-in-one interactive platform combining 6-DOF manipulation, real-time color detection, wireless IoT control, and mobile autonomy into one educational device.

EducationalPlatform
IoT ControlBlynk App

The Matrix Minds Robotic Arm is a 6-DOF, 3D-printed robotic arm mounted on a mobile four-wheeled platform. Designed as a comprehensive educational unit, it covers robotics, automation, embedded systems, and IoT in a single hands-on device.

Controlled by an ESP32 microcontroller and six servo motors, the system uses a TCS3200 color sensor to detect and classify objects by color, automatically routing them to designated sorting zones. All operations are monitored and commanded through a custom Blynk mobile application.

System Architecture

How It All Connects

Five integrated units working in concert to deliver real-time automated sorting and wireless manual control.

Object Placed

Colored object positioned before TCS3200 sensor on robot platform

Color Detection

TCS3200 filters RGB channels, outputs frequency signals

ESP32 Processing

Reads RGB frequencies, compares against thresholds, identifies color

PCA9685 Signal

ESP32 sends I2C commands, PCA9685 generates precise PWM for servos

Pick & Sort

Arm moves to object, gripper closes, carries to designated color zone

Return Home

Arm resets to default position, system awaits next detection cycle

System Units
⚡ Power: 12V Battery → LM2596 (6.5V) → AMS1117 (3.3V) / 7805 (5V)
📥 Input: TCS3200 Color · HC-SR04 Distance · Blynk App Commands
🧠 Processing: ESP32 reads, decides, dispatches all commands
🎮 Control: PCA9685 (arm servos) + L298N (platform motors)
⚙️ Output: 6× Servo (arm motion) + 4× DC Motor (platform movement)
